Bootcamp 2020: 2D Solar System Animation

This is the first project that I've created during the first 2 weeks at Bootcamp. Over those two weeks, I've learnt many new things about programming and tried to implement as many things as I could in my own work. My project is based around one of the lectures about classes and objects. My project is about a 2D visual representation of our solar system. I used a planet-class to store all the code related to each planet, so and how they will move and where they would be created. Each planet moves along its own orbital path at various speeds. The year counter of each planet increased every time it completes one whole orbit. I've also made it user interactive by adding a key that the user can press to show and hide planet names.
